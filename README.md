# ✨ Stardust - 星尘回忆

> 一个基于手势交互的 3D 粒子相册体验，用星空承载你的回忆

## 🌟 特性

- 🖐️ **手势控制** - 通过摄像头识别手势，无需触摸即可交互
- ✨ **粒子动画** - 2000+ 粒子形成心形、玫瑰、星环等形态
- 🌠 **流星雨** - 动态流星划过星空
- 📷 **照片展示** - 上传照片在星空中漂浮展示
- 💬 **祝福语** - 中英文艺术字体祝福卡片
- 🎵 **背景音乐** - 随机播放舒缓音乐
- 🎨 **暮光金紫** - 高级马卡龙配色方案
