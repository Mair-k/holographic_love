# Stardust Memories (私人星尘回忆) 🌸

这是一个基于 WebGL 的沉浸式 3D 照片画廊。它将我的照片转化为漂浮的星尘，完全通过摄像头捕捉手势来控制。

没有复杂的界面，只有手势、音乐和回忆。

## ✨ 主要功能

- **星空氛围**：樱花粉紫色的星云环境，带有流星和辉光特效。
- **手势交互**：
  - **精准倒计时**：通过伸出手指数量 (3 -> 2 -> 1) 精确控制启动。
  - **全景旋转**：握紧拳头移动，可 360° 自由旋转整个星系。
  - **细节查看**：捏合手指即可抓取并放大单张照片。
- **动态展示**：照片在心形队列和漫天散布之间自由切换。

## 🎮 操作指南

| 阶段 | 手势 | 动作说明 |
| :--- | :--- | :--- |
| **启动** | 🖐️ **手指计数** | 对着摄像头依次伸出 **3根、2根、1根** 手指触发启动。 |
| **浏览** | 🖐️ **张开手掌** | 将照片散布到空中，随光标移动视角。 |
| **旋转** | ✊ **握拳移动** | **握紧拳头**并上下左右移动，旋转整个场景。 |
| **查看** | 👌 **捏合手指** | 选中光标下的照片，捏合可拉近放大。 |
| **返回** | 🤚 **松开手指** | 松开捏合，照片自动归位。 |

## 🚀 如何运行

由于涉及摄像头权限，需要在本地服务器环境下运行：

1. **下载代码**
2. **启动服务** (任选一种方式):
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server`
   - VS Code: 使用 "Live Server" 插件
3. **访问**: 打开浏览器访问 `http://localhost:8000`

## 🛠️ 技术栈

- **Three.js**: 3D 场景渲染与粒子系统
- **MediaPipe**: 实时手部关键点追踪
- **HTML5/CSS3**: 界面与布局
